# RentFlow Backend

This folder contains the Java Spring Boot backend application for the RentFlow platform.

## 🛠 Technology Stack
- **Language**: Java 17+
- **Framework**: Spring Boot 3.x
- **Build Tool**: Maven
- **Security**: Spring Security (JWT-based token authentication)
- **Database**: PostgreSQL 15+
- **Migrations**: Flyway or Liquibase
- **Nomba API Integration**: RestTemplate / RestClient

## 📁 Source Layout
- `src/main/java`: Backend application logic.
- `src/main/resources`: Configuration properties, DB migrations (`db/migration/`), and application settings.
